คู่มือการใช้งาน CREATIVE TEAM

ระบบติดตามสถานะงานทีม (Google Sheets) และการแจ้งเตือนตอบกลับอัตโนมัติผ่าน LINE Bot

📌 1. ภาพรวมระบบ (System Overview)

CREATIVE TEAM ถูกพัฒนาขึ้นเพื่อบริหารจัดการและติดตามสถานะงานของสมาชิกในทีมทั้ง 7 คน (จ๊ะเอ๋, อุ้ม, กิ๊บ, เป้, โชกุล, ท้อป, โอม) โดยเชื่อมโยงตาราง Google Sheets ของทุกคนเข้ากับ LINE Bot และ Web App Dashboard เพื่อให้สามารถตรวจงานและเปลี่ยนสถานะได้อย่างรวดเร็ว ไร้รอยต่อ

📊 Google Sheets

ฐานข้อมูลหลักเก็บรายการงาน วันที่ส่ง และรายละเอียดลิงก์ของทีมทุกคน

📱 LINE Bot Notification

แจ้งเตือนงานรอตรวจแบบ Flex Message พร้อมปุ่มกดเปลี่ยนสถานะในแชท

💻 Web App Dashboard

หน้าเว็บติดตามสถานะงาน เรียบหรู คลีน สดใส พร้อมอัปเดตแบบ Auto-refresh

📱 2. การใช้งานผ่าน LINE Bot

  1. การแจ้งเตือนงานรอตรวจ (Sent to P'Aof):
    เมื่อพนักงานอัปเดตสถานะงานใน Google Sheet เป็น Sent to P'Aof ระบบจะส่งการ์ดแจ้งเตือนเข้ากลุ่ม LINE ทันที พร้อมแสดงชื่อชิ้นงาน ผู้รับผิดชอบ และวันที่ส่ง
  2. ปุ่มกดเปลี่ยนสถานะทันที:
    • มีปรับแก้ : กดเพื่อแจ้งว่างานชิ้นนี้มีจุดต้องแก้ไข สถานะใน Sheet จะเปลี่ยนเป็น "มีปรับแก้" ทันทีพร้อมตอบกลับเวลาไทยปัจจุบัน
    • Done : กดเมื่อตรวจผ่านแล้ว สถานะใน Sheet จะเปลี่ยนเป็น "Done" ทันที
  3. ระบบป้องกันการกดซ้ำ (Anti Double-Click):
    หากงานชิ้นนั้นถูกกดเปลี่ยนสถานะไปแล้ว (ไม่ว่าจะกดผ่าน LINE หรือผ่านหน้าเว็บ) หากมีคนเผลอกดปุ่มเดิมซ้ำ บอทจะแจ้งเตือนว่า "⚠️ ปุ่มนี้ถูกกดไปแล้วครับ!" เพื่อป้องกันข้อมูลสับสน

💻 3. การใช้งานผ่าน Web App Dashboard

  1. การกรองสถานะงาน (Filter Tabs):
    มีปุ่มสลับดูงานตามสถานะด้านบน: Sent to P'Aof และ มีปรับแก้ พร้อมแสดงตัวเลขจำนวนงานค้างแบบเรียลไทม์
  2. การดูรายละเอียดงาน:
    การ์ดงานแต่ละใบจะแสดงชื่องาน, ป้ายสถานะใต้ชื่องาน, วันเวลาส่งงาน, เจ้าของงาน และปุ่ม 🔗 เปิดดูรายละเอียดงาน สำหรับกดเปิดลิงก์ชิ้นงาน (เช่น Google Drive, Canva, Slide)
  3. ระบบดึงข้อมูลอัตโนมัติ (Silent Auto-Refresh):
    หน้าเว็บจะแอบดึงข้อมูลอัปเดตจาก Google Sheets ทุกๆ 10 วินาทีโดยอัตโนมัติ เพื่อให้เห็นสถานะล่าสุดอยู่เสมอโดยไม่ต้องกด F5 รีเฟรชหน้าเว็บ

⚙️ 4. การจัดการข้อมูลใน Google Sheets

ระบบอ่านข้อมูลจากคอลัมน์มาตรฐานดังนี้:

  • คอลัมน์ A (Status): สถานะงาน (Sent to P'Aof, มีปรับแก้, Done ฯลฯ)
  • คอลัมน์ E (Date): วันเวลาที่ส่งงาน
  • คอลัมน์ J (Job No.): รหัสงาน
  • คอลัมน์ L (Task Name): ชื่องาน/โปรเจกต์
  • คอลัมน์ M (Owner): ชื่อผู้รับผิดชอบงาน
  • คอลัมน์ N (Detail / Link): รายละเอียดงานหรือลิงก์ส่งงาน
